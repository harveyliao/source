Julia 集

在上一节中，我们展示了如何使用表达式 \[z_{n+1}=z_{n}^2+z_0.\] 生成 Mandelbrot 集。 这是二次递推方程 \begin{eqnarray}\label{julia} z_{n+1}=z_{n}^2+c \end{eqnarray} 的一个特例， 其中 $c$ 是一个固定的复数。用这个方程得到的集合称为 Julia 集。 事实上，几乎每个 $c.$ 都对应一个不同的 Julia 集。

与我们对 Mandelbrot 集所做的类似，我们得到一个复数序列 $z_n$，其中 $n=0,1,2,\ldots.$。 同样，点 $z_n$ 被称为构成 $z_0,$ 的轨道，而 Julia 集定义如下：

如果轨道 $z_n$ 未能逃逸到无穷远，则初始点 $z_0$ 称为属于填充 Julia 集。

Julia 集以法国数学家 Gaston Julia 命名， 他于 1915 年研究了它们的性质，并在 1918 年发表了著名的论文： Mémoire sur l'itération des fonctions rationnelles。 虽然 Julia 集现在与 (\ref{julia}) 中的二次多项式相关联，但 Julia 实际上对更一般表达式的迭代性质感兴趣，即 \[z^4 + \frac{z^3}{z-1} + \frac{z^2}{z^3 + 4 z^2 + 5} + c.\] 由方程 (\ref{julia}) 定义的 Julia 集可以呈现各种各样的形状，$c$ 的微小变化 可以极大地改变 Julia 集。1979 年，借助计算机，B. B. Mandelbrot 研究了 Julia 集并尝试对所有可能的形状进行分类， 从而产生了一个新的形状：Mandelbrot 集。

Mandelbrot 与 Julia 集的联系

由于 Mandelbrot 集的定义，Mandelbrot 集在给定点处的几何与对应 Julia 集的结构之间存在密切的对应关系。 换句话说，Mandelbrot 集构成了 Julia 集的一种索引。 Julia 集要么是连通的，要么是不连通的，从 Mandelbrot 集内部选取的 $c$ 值是连通的， 而从 Mandelbrot 集外部选取的值是不连通的。 不连通集通常称为尘埃，它们由孤立点组成，无论以何种分辨率查看都是如此。

在下面的小程序中探索 Mandelbrot 集与 Julia 集之间的关系。 将鼠标移到 Mandelbrot 集上以观察不同的 Julia 集。在不同区域放大或缩小。 打开控制菜单以更改迭代次数或选择特定的 $c.$ 值。
