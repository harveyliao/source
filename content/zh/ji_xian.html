极限

设函数 $f$ 在 $z_0.$ 的某个去心邻域内的所有点 $z_0.$ 处有定义。 $f (z)$ 当 $z$ 趋近于 $z_0$ 时的极限为数 $w_0,$，即 \begin{eqnarray}\label{limit001} \lim_{z\rightarrow z_0}f(z)=w_0 \end{eqnarray} 这意味着如果选取点 $z$ 足够接近 $z_0$ 但不等于它，则点 $z$ 可以任意接近 $z_0$。

形式上，表达式 (\ref{limit001}) 意味着对于每个（充分小的）$\varepsilon>0,$，存在 $\delta>0$ 使得

\begin{eqnarray}\label{limitcondition} \left|f (z) - w_0\right| \lt \varepsilon \quad \text{whenever}\quad 0 \lt \left|z - z_0\right| \lt \delta \end{eqnarray}

在映射 $w = f (z),$ 下，圆 $|z - z_0| = \delta$ 内所有去掉了 $z_0$ 的点都被映射到圆 $|w - w_0| = \varepsilon,$ 内的点，见图 1。极限仅在 $z$ 以任意方向趋近于 $z_0$ （即 $z \rightarrow z_0$）时才存在；此时这意味着 $w \rightarrow w_0$。


Limits are unique if they exist.
Suppose that $\lim_{z\rightarrow z_0}f(z)=w_0$ and $\lim_{z\rightarrow z_0}f(z)=w_1$ with $w_0\neq w_1.$ Let $2\varepsilon =|w_0-w_1|,$ so that $\varepsilon >0.$ There is a $\delta>0$ such that $0\lt|z-z_0|\lt\delta$ implies that $|f(z)-w_0|\lt\varepsilon$ and $|f(z)-w_1|\lt\varepsilon.$ Choose a point $z$ different to $z_0$ (because $f$ is defined in a deleted neighborhood of $z_0$). Then, using the triangle inequality, we have
$$2 \epsilon = |w_0-w_1| = |w_0 - f(z) + f(z) - w_1| \leq |w_0-f(z)|+|f(z)-w_1|\lt2\varepsilon.$$
But this is a contradiction. Thus $w_0=w_1.$

例 1： 证明 $$\lim_{z\rightarrow z_0}z^2=z_0^2.$$

讨论： 考虑 $\delta \leq 1.$。所以 $0\lt\left| z-z_0\right|\lt\delta$ 意味着

\begin{eqnarray*} \left| z^2-z_0^2\right|&=&\left| z-z_0\right|\left| z+z_0\right|\\ &\lt& \delta \left| z+z_0\right|\\ &\lt& \delta \left| z-z_0+2z_0\right|\\ &\lt& \delta \big( \left| z-z_0\right|+2\left|z_0\right|\big) \\ &\lt&\delta \big(1+2\left|z_0\right|\big). \end{eqnarray*}
Now, take $\delta = 1 $ or $\delta=\dfrac{\varepsilon}{1+2\left|z_0\right|},$ whichever is smaller. Then $0\lt\left| z-z_0\right|\lt\delta$ implies that $$\left| z^2-z_0^2\right|\lt\varepsilon.$$ This means that $$\left| z^2-z_0^2\right|\lt\varepsilon \quad\text{whenever}\quad 0\lt\left| z-z_0\right|\lt\delta$$ where $\delta=\min\left\{1,\dfrac{\varepsilon}{1+2\left|z_0\right|}\right\}.$

现在我们可以着手写出形式化的证明。

证明： 令 $\varepsilon>0.$。选取 $\delta=\min\left\{1,\dfrac{\varepsilon}{1+2\left|z_0\right|}\right\}$ 使得 $0\lt\left| z-z_0\right|\lt\delta.$。因此 $$ \left| z^2-z_0^2\right|\lt\varepsilon. \quad \blacksquare $$

复变函数 $f(z)$ 的极限与 两个实变量的实值函数 $g(x,y).$ 的极限之间存在联系。 后一种类型在微积分中研究过，我们可以利用它们的定义和性质，如下所示：

Consider $f(z)=u(x,y)+iv(x,y)$ and $z_0=x_0+iy_0,$ $w_0=u_0+iv_0.$ Then \begin{eqnarray}\label{limitpart01} \lim_{z\rightarrow z_0}f(z)=w_0 \end{eqnarray} if and only if \begin{eqnarray}\label{limitpart02} \lim_{\left(x,y\right) \rightarrow \left(x_0,y_0\right)}u\left(x,y\right)=u_0\quad \text{and}\quad \lim_{\left(x,y\right) \rightarrow \left(x_0,y_0\right)}v\left(x,y\right)=v_0 \end{eqnarray}
First, let us assume that limit (\ref{limitpart01}) holds. Thus, for each positive number $\varepsilon,$ there is a positive number $\delta$ such that \begin{eqnarray}\label{limpart05} |(u + iv) -(u_0 + iv_0)| \lt \varepsilon \end{eqnarray} whenever \begin{eqnarray}\label{limpart06} 0 \lt |(x + iy) - (x_0 + iy_0)| \lt \delta \end{eqnarray} But
\begin{eqnarray*} |u-u_0|&\leq &|(u-u_0) + i(v -v_0)|\;=\;|(u + iv) - (u_0 + iv_0)|,\\ |v-v_0|&\leq &|(u-u_0) + i(v -v_0)|\;=\;|(u + iv) - (u_0 + iv_0)| \end{eqnarray*}
and
$$|(x+iy)-(x_0+iy_0)|=|(x-x_0)+i(y-y_0)|=\sqrt{(x-x_0)^2+(y-y_0)^2}$$
Hence it follows from inequalities (\ref{limpart05}) and (\ref{limpart06}) that
$$|u-u_0|\lt\varepsilon\quad \text{and}\quad |v-v_0|\lt\varepsilon$$
whenever
$$0\lt\sqrt{(x-x_0)^2+(y-y_0)^2}\lt\delta$$
which proves that limits (\ref{limitpart02}) hold.

现在，我们假设极限 (\ref{limitpart02}) 成立以得到极限 (\ref{limitpart01})。 极限 (\ref{limitpart02}) 告诉我们，对于每个正数 $\varepsilon,$，存在正数 $\delta_1$ 和 $\delta_2$ 使得

\begin{eqnarray}\label{limpart03} |u-u_0|\lt \frac{\varepsilon}{2}\quad\text{whenever}\quad 0\lt\sqrt{(x-x_0)^2+(y-y_0)^2}\lt\delta_1 \end{eqnarray}
和
\begin{eqnarray}\label{limpart04} |v - v_0| \lt \frac{\varepsilon}{2}\quad\text{whenever}\quad 0\lt\sqrt{(x-x_0)^2+(y-y_0)^2}\lt\delta_2 \end{eqnarray}
令 $\delta$ 是小于 $\delta_1$ 和 $\delta_2.$ 的任何正数。因为
$$|(u + iv) - (u_0 + iv_0)| = |(u - u_0) + i(v - v_0)| \leq |u - u_0| + |v - v_0|$$
和
$$\sqrt{(x-x_0)^2+(y-y_0)^2}=|(x-x_0)+i(y-y_0)|\leq |(x+iy)-(x_0+iy_0)|$$
由陈述 (\ref{limpart03}) 和 (\ref{limpart04}) 可得
$$|(u + iv) -(u_0 + iv_0)| \lt\frac{\varepsilon}{2}+\frac{\varepsilon}{2}=\varepsilon$$
每当
$$0 \lt |(x + iy) - (x_0 + iy_0)| \lt \delta.$$
即极限 (\ref{limitpart01}) 成立，这完成了定理的证明。

例 2： 证明 $$\lim_{z\rightarrow 0}\dfrac{z}{\overline{z}}$$ 不存在。

解： 假设极限存在。那么我们可以通过令点 $z = x+iy$ 以任何方式趋近于原点来计算它。 然而，当 $z = x+i0$ 是实轴上的非零点时 $$f(z)=\frac{x+i0}{x-i0}=1;$$ 而当 $z=0+iy$ 是虚轴上的非零点时， $$f(z)=\frac{0+iy}{0-iy}=-1.$$

因此，如果我们令 $z$ 沿实轴趋近于原点，我们发现极限为 $1.$。 另一方面，如果沿虚轴趋近，我们得到的极限是 $-1,$， 这产生矛盾，因为极限是唯一的。因此我们必须断定极限 $$\lim_{z\rightarrow 0}\dfrac{z}{\overline{z}}$$ 不存在。

除了计算特定的极限外，**定理 2** 也是一个重要的理论工具，使我们能够从实极限的性质推导出复极限的许多性质。 下面的定理给出了这一过程的一个例子。

假设 \begin{eqnarray} \lim_{z\rightarrow z_0}f(z)=w_1\quad \text{and}\quad \lim_{z\rightarrow z_0}g(z)=w_2\label{limithypo} \end{eqnarray} 则 \begin{eqnarray} \lim_{z\rightarrow z_0}\big[c\cdot f(z)\big] &=& c\cdot \lim_{z\rightarrow z_0} f(z) \text{ with } c\in\C, \label{limitconst}\\ \lim_{z\rightarrow z_0}\big[f(z)+g(z)\big]=w_1+w_2,\label{limitsum}\\ \lim_{z\rightarrow z_0}\big[f(z)g(z)\big]=w_1w_2;\label{limitmult} \end{eqnarray} 且如果 $w_2\neq 0,$ \begin{eqnarray}\label{limitdiv} \lim_{z\rightarrow z_0}\frac{f(z)}{g(z)}=\frac{w_1}{w_2}. \end{eqnarray}

For proving property (\ref{limitsum}), we must show that for any $\varepsilon >0$ we can find $\delta >0$ such that
\begin{eqnarray*} \left| \left(f(z)+g(z)\right)-\left(w_1+w_2\right)\right|\lt\varepsilon \quad \text{when}\quad 0\lt\left| z-z_0\right|\lt\delta \end{eqnarray*}
By hypothesis, given $\varepsilon >0$ we can find $\delta_1 >0$ and $\delta_2>0$ such that
\begin{eqnarray*} \left| f(z)-w_1\right|\lt\varepsilon/2 \quad \text{when}\quad 0\lt\left| z-z_0\right|\lt\delta_1 \\ \left| g(z)-w_2\right|\lt\varepsilon/2 \quad \text{when}\quad 0\lt\left| z-z_0\right|\lt\delta_2 \end{eqnarray*}

于是我们有

\begin{eqnarray*} \left| \left(f(z)+g(z)\right)-\left(w_1+w_2\right)\right| &=&\left| \left(f(z)-w_1\right)+\left(g(z)+w_2\right)\right| \\ &\leq &\left| f(z)-w_1\right|+\left| g(z)-w_2\right|\\ &\leq & \varepsilon/2+\varepsilon/2=\varepsilon \end{eqnarray*}
因此，
\begin{eqnarray*} \left| \left(f(z)+g(z)\right)-\left(w_1+w_2\right)\right|\lt\varepsilon \quad \text{when}\quad 0\lt\left| z-z_0\right|\lt\delta \end{eqnarray*}
其中 $\delta$ 取 $\delta_1$ 和 $\delta_2.$ 中较小者。

现在我们将证明性质 (\ref{limitmult})。 首先，利用三角不等式，我们写

\begin{eqnarray*} \left|f(z)g(z) - w_1w_2\right|& \leq & \left|f(z)g(z) - f(z)w_2\right| + \left|f(z)w_2 - w_1w_2\right|\\ &=& \left|f(z)\right|\left|g(z) - w_2\right|+ \left|f(z) - w_1\right|\left|w_2\right| \end{eqnarray*}
为估计每一项，我们选取 $\delta_1\gt 0$ 使得 $0\lt \left|z-z_0\right|\lt \delta_1$ 蕴含 $\left|f(z)- w_1\right|\lt 1,$ 从而 $\left|f(z)\right|\lt \left|w_1\right|+1,$，因为 $$\left|f(z) - w_1\right|\geq \left|f(z)\right|- \left| w_1\right|.$$ 给定 $\varepsilon\gt 0,$，设 $\delta_2$ 和 $\delta_3$ 使得 $0\lt \left|z-z_0\right|\lt \delta_2$ 蕴含 $$\left|f(z)-w_1\right|\lt \frac{\varepsilon}{2\left(|w_2|+1\right)}$$ 以及 $0\lt \left|z-z_0\right|\lt \delta_3$ 蕴含 $$\left|g(z)-w_2\right|\lt \frac{\varepsilon}{2\left(|w_1|+1\right)}.$$ 因此我们可以选取 $\delta = \min\left\{\delta_1,\delta_2,\delta_3\right\}.$ 如果 $0\lt \left|z-z_0\right|\lt \delta,$，则
\begin{eqnarray*} \left|f(z)g(z) - w_1w_2\right|& \leq & \left|f(z)\right|\left|g(z) - w_2\right|+ \left|f(z) - w_1\right|\left|w_2\right|\\ &\lt & \frac{\varepsilon}{2\left(|w_1|+1\right)}\left|f(z)\right| + \frac{\varepsilon}{2\left(|w_2|+1\right)}\left|w_2\right|\\ &\lt & \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon. \end{eqnarray*}
因此，$\ds \lim_{z\rightarrow z_0}\big[f(z)g(z)\big]=w_1w_2$ 得证。

性质 (\ref{limitconst}) 和 (\ref{limitdiv}) 留作练习。
