含分支切割的函数的积分

当我们考虑多值函数时，围道积分中的路径可能包含被积函数分支切割上的一个点。接下来的两个例子说明了这一点。

例 1

设 $C$ 是从 $z_0=3$ 到 $z_1=-3.$ 的半圆路径。即 $z(\theta) = 3 e^{i\theta},$ 其中 $0\leq \theta \leq \pi.$ 这里我们想要求积分 \begin{eqnarray}\label{ex-01} I = \int_C z^{1/2} dz. \end{eqnarray} 为此，我们需要选择多值函数 $z^{1/2}.$ 的一个特定分支。例如，我们将使用主支 \begin{eqnarray} |z|> 0, \; -\pi \lt \textbf{Arg}\,(z)\lt \pi. \end{eqnarray}

在这种情况下，注意虽然 $z^{1/2}= \exp\left(\frac{1}{2}\text{Log}\, z\right)$ 的主支 在围道 $C,$ 的端点 $C,$ 处没有定义，但积分 $I$ 仍然存在。

使用下面的小程序探索给定围道 $C.$ 的 $C.$ 值。 只需将点 $z_0$ 和 $z_1$ 拖到相应的值。 您也可以选择其他围道并探索当它们穿过分支切割 $\{z\,:\, x \leq 0 \text{ and } y =0\}.$ 时会发生什么。

正如您已经想到的，积分 (\ref{ex-01}) 存在是因为被积函数在 $C.$ 上是 分段连续的。 为确认这一点，注意到当 $z(\theta)= 3e^{i\theta},$ 时 $$ f\left(z(\theta)\right)= \exp\left[\frac{1}{2} \left(\ln 3 + i \theta\right)\right] = \sqrt{3}e^{i\theta/2}. $$

函数的实部和虚部在 $\theta = \pi $ 处的左极限

$\theta = \pi $
存在。即
\begin{eqnarray*} \lim_{\theta \rightarrow \pi-}-3\sqrt{3} \sin \frac{3\theta }{2}= 3\sqrt{3} \quad \text{ and } \quad \lim_{\theta \rightarrow \pi-} 3\sqrt{3} \cos \frac{3\theta }{2} = 0.\\ \end{eqnarray*}
这意味着当 $f\left(z(\theta)\right)z'(\theta)$ 在 $3\sqrt{3}.$ 处的值定义为 $\theta = \pi $ 时，它在闭区间 $3\sqrt{3}.$ 上连续。 因此
\begin{eqnarray*} I &=& \int_C f\left(z(\theta)\right)z'(\theta)\, d\theta= 3\sqrt{3} i \int_0^{\pi} e^{i\theta /2} d\theta = 3\sqrt{3} i\Bigg[ \bigg.\frac{2}{3i}e^{i3\theta /2}\bigg|_{0}^{\pi} \Bigg] \\ &=& 3\sqrt{3} i\Bigg[ -\frac{2}{3i}\left( 1+i\right)\Bigg]= -2\sqrt{3}(1+i). \end{eqnarray*}

练习 1： 求围道 $-\pi\leq \theta \leq \pi.$ 其中 $C: z(\theta)= e^{i\theta},$ 的 $-\pi\leq \theta \leq \pi.$。 您可以使用小程序确认您的结果。

注： 注意，对于任何不与分支切割 $\{z\,:\, x \leq 0 \text{ and } y = 0\}.$ 相交的圆，都有 $\{z\,:\, x \leq 0 \text{ and } y = 0\}.$。为什么？

例 2

考虑主支

\begin{eqnarray}\label{br} f(z)=z^{i} = \exp \left[ i \,\text{Log } z \right]\; \text{ with }\; |z|>0,\, -\pi\lt \text{Arg } z \lt \pi \end{eqnarray}
且 $C$ 是从 $z=-1$ 到 $z=1$ 的上半圆；即 $z(t)=-e^{-i \pi t}$ 其中 $0 \leq t \leq 1.$

不难验证 \begin{eqnarray}\label{ex-02} I=\int_C z^{i}\,dz &=& \frac{1+e^{-\pi}}{2}\,(1-i). \end{eqnarray} 使用下面的小程序来确认这一点。您也可以分析其他围道的 $\int_C z^idz$。

注： 注意，对于任何不与分支切割 $\{z\,:\, x \leq 0, y = 0\}.$ 相交的圆，都有 $\{z\,:\, x \leq 0, y = 0\}.$。为什么？

一般地，我们可以计算从 $z=1$ 到 $z=-1$ 且位于实轴上方的任意围道 $z=-1$ 的 $z=1$。 我们只需要找到 $z^i.$ 的一个原函数。 不幸的是，我们不能使用 (\ref{br}) 中定义的主支，因为该分支甚至在 $z=-1.$ 处都没有定义。 但被积函数可以替换为分支

$$z^{i} = \exp \left[ i \,\text{log } z \right]\; \text{ with }\; |z|>0,\, -\frac{\pi}{2}\lt \text{arg } z \lt \frac{3\pi}{2}.$$
因为它沿 $C.$ 与被积函数一致。

利用这个新分支的原函数，我们得到

\begin{eqnarray}\label{value} \int_{-1}^{1} z^i dz&=& \Bigg.\frac{z^{i+1}}{i+1}\Bigg|_{-1}^{1} = \frac{1+e^{-\pi}}{2}\,(1-i), \end{eqnarray}
这与 (\ref{ex-02}) 的值相同。

练习 2： 求 $\bigg.\dfrac{z^{i+1}}{i+1}\bigg|_{-1}^{1} $ 以确认 (\ref{value}) 的值。
