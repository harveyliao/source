复平面的拓扑

邻域

复数 $z_0$ 的一个 $z_0$ 邻域，也称为 开球或开圆盘，由所有满足条件的点 $z$ 组成，这些点位于以 $z_0$ 为圆心、半径为 $\varepsilon>0$ 的圆内部（不在圆周上），其表达式为

\begin{eqnarray} B_{\varepsilon}(z_0)=\{z:|z-z_0| \lt \varepsilon\}. \end{eqnarray}
$z_0$ 的闭 $z_0$ 邻域 表示为
\begin{eqnarray} \overline{B}_{\varepsilon}(z_0)=\{z:|z-z_0|\leq \varepsilon\}. \end{eqnarray}
最后，$z_0,$ 的去心 $z_0,$ 邻域， 也称为去心球或去心圆盘，表示为
\begin{eqnarray} B_{\varepsilon}(z_0)\setminus \{z_0\}=\{z:0 \lt |z-z_0| \lt \varepsilon\}. \end{eqnarray}
图 1 展示了以下例子的几何表示：

  1. $B_{1}(0)=\{z:|z|\lt 1\}$
  2. $\overline{B}_{\frac{7}{8}}\left(-1-\sqrt{2}i\right)=\left\{z:|z-\left(-1-\sqrt{2}i\right)|\leq \frac{7}{8}\right\}$
  3. $\overline{B}_{\frac{1}{2}}\left(2+\sqrt{3}i\right)\setminus\left\{2+\sqrt{3}i\right\}=\left\{z:0<|z-\left(2+\sqrt{3}i\right)|\leq \frac{1}{2}\right\}$

点的分类

点 $z_0$ 称为集合 $S\subset \mathbb C$ 的内点，如果存在 $z_0$ 的一个邻域 包含于 $S$ 中。 点 $S$ 称为集合 $S.$ 的外点，如果存在 $z_0$ 的一个邻域 与 $S.$ 不相交。 如果 $S$ 既不是 $S.$ 的内点也不是外点，则称其为边界点。 $S.$ 的所有边界点的集合记为 $$ S_1=B_1(0),\; S_2=\overline{B}_{\frac{7}{8}}\left(-1-\sqrt{2}i\right) \text{ and } S_3=\overline{B}_{\frac{1}{2}}\left(2+\sqrt{3}i\right)\setminus \left\{2+\sqrt{3}i\right\}. $$

本文中将使用以下记号：

  1. $\text{Int } S=\left\{z: z \text{ is an interior point of } S \right\}$
  2. $\text{Ext } S=\left\{z: z \text{ is an exterior point of } S \right\}$
  3. $\partial S=\left\{z: z \text{ is a boundary point of } S \right\}$

考虑前面的邻域例子，令 $S_1$ $\text{Int } S_1=B_1(0)$ 那么：

对于 $S_2$，有：

  • $\text{Int } S_2=B_{\frac{7}{8}}\left(-1-\sqrt{2}i\right)$
  • $\text{Ext } S_2=\left\{z:|z-\left(-1-\sqrt{2}i\right)|>\frac{7}{8}\right\}$
  • $\partial S_2=\left\{z:| z-\left(-1-\sqrt{2}i\right)| =\frac{7}{8}\right\}$

最后，对于 $S_3$，有：

  • $\text{Int } S_3=B_{\frac{1}{2}}\left(2+\sqrt{3}i\right)\setminus \{2+\sqrt{3}i\}$
  • $\text{Ext } S_3=\left\{z:|z-\left(2+\sqrt{3}i\right)|>\frac{1}{2}\right\}$
  • $\partial S_3=\left\{z:|z-\left(2+\sqrt{3}i\right)|=\frac{1}{2}\right\}\cup\{2+\sqrt{3}i\}$

拓扑空间

集合 $S$ 称为开集，如果对于每个 $z\in S,$ 都存在一个 $\varepsilon>0$ 的邻域 完全包含于 $\text{Int } S= S.$ 中。 集合 $S$ 称为闭集，如果它在 $\mathbb C$ 中的补集是开集。

集合 $\mathbb C,$ 连同其子集的收集 $\tau =\{S\subseteq \mathbb C: S \text{ is open} \}$ 构成一个拓扑空间，如果满足以下条件：
- $\left(\mathbb C, \tau\right).$ 和 $\left(\mathbb C, \tau\right)$ 属于 $\emptyset$
- $\mathbb C$ 中任意多个集合的并集仍在 $S_1$ 中。
- $S_2$ 中有限多个集合的交集仍在 $S_1\bigcap S_2.$ 中。

注： 拓扑空间的技术定义有点超出本文的范围。 然而，需要指出的是集合 $\mathbb C$ 的收集满足上述所有三个条件， 因此 $$B_{\varepsilon}(z)\subset S.$$ 是一个拓扑空间。

集合 $S$ 的闭包是包含 $S$ 的所有闭集的交集， 记为 $$\partial S\subseteq S.$$ 因此，$S.$ 是包含 $S$ 的最小闭集。

一般来说，如果一个集合不能表示为两个不相交的非空开子集的并集，则称其为连通的。 在复平面中，一个开集是连通的当且仅当其中任意两点都可以用完全位于该集合内的折线连接。


例如，注意开集 $|z| \lt 1$ 是连通的，因为其内部任意两点都可以用一条折线连接。



一个非空且连通的开集称为区域。 可将其视为由内部点组成且任意两点可用折线连接的集合。


练习： 在复平面中画出集合 $S$ 并指出其中的点 $$\overline{S}=S\cup \partial S.$$

  • $-1\lt \text{Re(z)} \lt 1$
  • $|z|>1$
  • $2\le |z - 3+4i |\le 5$
